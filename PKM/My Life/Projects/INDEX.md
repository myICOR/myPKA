# Projects - Index

Bounded efforts with a finish line — a set of deliverables you cross off when finished. A Project is the time-bound vehicle that moves a Goal forward. It has a scope; a Habit has only a tempo.

## Active files

- [[side-project-mvp]] — seeded course sample. The canonical shape every Project file follows.

## Naming

Kebab-case slug, no date prefix. One file per project.

See [[GL-001-file-naming-conventions]].
