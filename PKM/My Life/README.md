# My Life

This is the part of my myPKA that holds the five life concepts I actually work with: Topics, Habits, Goals, Projects, and Key Elements. Each has its own flat folder. Everything connects through Obsidian-style `[[wikilinks]]` rather than nested hierarchies.

A Topic might link to a Habit. A Habit feeds a Goal. A Goal sits inside a Key Element. A Project is the time-bound push that moves a Goal forward. The folder structure is dumb on purpose. The links carry the meaning.
